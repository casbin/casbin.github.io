온라인 편집기

온라인 편집기(https://casbin.org/en/editor/)를 사용해서 브라우저에서 Casbin 모델(model)과 정책(policy)을 작성할 수 있습니다. 프로그래밍 언어를 위한 통합 개발 환경(IDE) 처럼 구문 강조(syntax highlighting) 및 코드 자동 완성(code completion) 기능을 제공합니다.
